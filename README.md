🛒 CartHub - Multi Vendor E-Commerce Website

CartHub is a complete Multi Vendor E-Commerce Web Application developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.
The platform allows users to shop products online, sellers to manage their stores, and admins to control the entire system through a powerful dashboard.

🚀 Features
👤 User Features

• ✅ User Registration & Login
• 🔐 Forgot Password with OTP Verification
• 🔍 Product Search System
• 📦 Product Details Page
• 🛒 Add to Cart Functionality
• 💳 Checkout System
• 📑 Order Placement
• 🚚 Order Tracking
• 🧾 Invoice Generation
• ⭐ Product Reviews & Ratings
• 👤 User Profile Management
• 📋 My Orders Section

🏪 Seller Features

• 📊 Seller Dashboard
• ➕ Add New Products
• 📦 Inventory Management
• 📑 Order Management
• 📈 Sales Analytics
• 👤 Seller Profile Management
• 💰 Payout Processing

🛠️ Admin Features

• 🔐 Secure Admin Login
• 👥 Manage Users
• 📦 Manage Products
• 📊 Website Analytics
• 🖼️ Homepage Slider Management
• 📤 Export User Data
• 🛡️ Full Platform Monitoring

🔄 Project Flow
🔐 1. Authentication System

The project starts with a secure authentication system where users can:

• 👤 Register a new account
• 🔑 Login securely
• 📧 Reset forgotten passwords using OTP verification through email

📁 Folder:

auth/

🛍️ 2. User Module

After successful login, users can:

• 🔍 Browse products
• 📦 View product details
• 🛒 Add products to cart
• 💳 Checkout products
• 📑 Place orders
• 🚚 Track order status
• 🧾 Download invoices
• ⭐ Give ratings & reviews

📁 Folder:

user/

🛒 3. Cart & Checkout System

The cart system allows users to:

• ➕ Add products to cart
• ✏️ Update quantities
• ❌ Remove products
• 💳 Proceed to checkout
• ✅ Place orders successfully

📁 Main File:

cart_action.php

🏪 4. Seller Module

Sellers can manage their own products and store:

• 📦 Add/Edit/Delete products
• 📊 View analytics
• 📑 Manage customer orders
• 💰 Handle payouts
• 👤 Update seller profile

📁 Folder:

seller/

🛠️ 5. Admin Panel

The admin has full control over the platform:

• 👥 Manage all users
• 📦 Control products
• 📊 Monitor analytics
• 🖼️ Manage homepage content
• 📤 Export data
• 🛡️ Monitor complete system activities

📁 Folder:

admin/
